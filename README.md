# 🛡️ 逻辑指挥官 - 算力基建监控哨兵 v3.0

A股多股票实时监控仪表盘 - 专业版

## ✨ v3.0 新功能

### 📊 多股票监控
- 支持同时监控多只股票
- 预设 8 只热门股票（可扩展）
- 支持自定义添加任意 A 股

### 🎯 多维度报警
- **价格报警**: 支撑位/压力位突破提醒
- **涨跌幅报警**: 自定义涨跌幅阈值
- **成交量报警**: 放量异动提醒

### 📱 微信推送
- 集成 PushPlus 免费推送服务
- 报警信息实时推送到微信
- 防重复推送机制

### 📰 新闻收集
- 自动获取股票相关新闻
- 财经资讯实时更新

## 🚀 快速开始

### 在线访问
部署到 Streamlit Cloud 即可使用

### 本地运行
```bash
git clone https://github.com/shine197712-johnson/stock-dashboard.git
cd stock-dashboard
pip install -r requirements.txt
streamlit run app.py
```

## 📱 配置微信推送

1. 访问 [pushplus.plus](https://www.pushplus.plus/)
2. 微信扫码登录
3. 复制个人 Token
4. 粘贴到应用侧边栏的 "PushPlus Token" 输入框
5. 点击 "测试" 验证是否成功
