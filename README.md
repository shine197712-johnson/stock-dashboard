# 🛡️ 逻辑指挥官 - 算力基建监控哨兵 v3.1

A股全市场实时监控仪表盘

## ✨ v3.1 功能

### 🔍 全A股搜索
- 支持 **5000+** 只 A 股实时搜索
- 输入股票代码（如 301165）或名称（如 锐捷）即可搜索
- 搜索结果一键添加到监控列表

### 🎯 5维度报警
每只股票可独立配置以下报警：

| 维度 | 说明 |
|------|------|
| 💰 价格区间 | 设置价格上限/下限 |
| 📊 涨跌幅 | 设置涨幅/跌幅阈值 |
| 📦 成交量 | 成交量超过阈值报警 |
| 💵 成交额 | 成交额超过阈值报警 |
| 🔄 换手率 | 换手率超过阈值报警 |

### 📱 微信推送
- 集成 PushPlus 免费服务
- 触发报警自动推送微信
- 防重复推送

## 🚀 使用方法

1. 左侧搜索框输入股票代码或名称
2. 点击 ➕ 添加到监控
3. 展开"报警设置"配置各项阈值
4. 配置 PushPlus Token 接收微信通知

## 📱 获取 PushPlus Token

1. 访问 [pushplus.plus](https://www.pushplus.plus/)
2. 微信扫码登录
3. 复制 Token 粘贴到应用中

## 本地运行

```bash
pip install -r requirements.txt
streamlit run app.py
```
