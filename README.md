## 🚀 部署方法

### 更新现有部署

1. 在 GitHub 仓库中替换以下文件:
   - `app.py` - 全部替换
   - `requirements.txt` - 保持不变

2. Streamlit Cloud 会自动检测更新并重新部署

3. 等待 2-3 分钟后刷新页面

### 全新部署

1. Fork 或克隆此仓库到 GitHub
2. 登录 [Streamlit Cloud](https://streamlit.io/cloud)
3. 点击 "New app"
4. 选择仓库和 `app.py` 文件
5. 点击 "Deploy"

## 🖥️ 本地运行

```bash
# 安装依赖
pip install -r requirements.txt

# 运行应用
streamlit run app.py
```
